BUY LATER — Installation & Usage Guide
======================================

Buy Later is a cooling-off app: photograph something you want, let it sit,
then decide. Under $50 → review after 7 days. $50 and over → wait 14 days.

It is built as an installable web app (PWA). On Android it installs to your
home screen and runs full-screen like any other app, fully offline.

------------------------------------------------------------
PRIVACY & SECURITY (what "no data leaves the device" means)
------------------------------------------------------------
• The app makes ZERO network requests. There is no fetch/XHR call to any
  server anywhere in the code. Your items, photos, prices and notes are
  stored only in your phone's local browser database (IndexedDB).
• The bundled service worker actively BLOCKS any cross-origin request, so
  even a future mistake can't leak data off-device.
• No accounts, no sign-in, no analytics, no ads, no third-party scripts.
• Photos are downscaled and kept on-device; the optional photo scan (OCR)
  runs locally — the text recognition never uploads your image.
• "Export backup" writes a plain JSON file YOU control. Nothing is sent.

You can verify all of the above: every file is plain text you can open and
read. Search the code for "http" — you'll find none pointing outward.

INSTALL ON ANDROID — EASIEST METHOD (recommended)
------------------------------------------------------------
A PWA must be served over a local web address to install. Pick ONE:

OPTION 1 — Host the folder (simplest, one-time)
  1. Put the whole "buylater" folder on any static host you trust, e.g.
     GitHub Pages, Netlify Drop (netlify.com/drop), or Cloudflare Pages.
     (These only HOST the files; the app still never sends them your data.)
  2. On your Android phone, open the resulting https:// link in Chrome.
  3. Tap the ⋮ menu (top-right) → "Add to Home screen" / "Install app".
  4. Confirm. "Buy Later" now appears in your app drawer and works offline,
     even in airplane mode, from then on.

OPTION 2 — Run a tiny local server from the phone (fully offline install)
  1. Copy the "buylater" folder to your phone's storage.
  2. Install a local-server app such as "Simple HTTP Server" or "Termux".
     - Termux: run  `pkg install python`  then in the folder
       `python -m http.server 8080`
  3. In Chrome go to  http://localhost:8080  → ⋮ → "Add to Home screen".
  After install it runs offline; you can stop the server.

OPTION 3 — Just open it (no install, still works)
  1. Copy the folder to the phone.
  2. Open index.html in Chrome via a file manager.
  Note: file:// mode disables the service worker, so it won't show the
  "Install" prompt and reminders are limited — but you can still add and
  review items. Use Option 1 or 2 for the real installed experience.

------------------------------------------------------------
ENABLE PHOTO AUTO-SCAN (optional, stays 100% offline)
------------------------------------------------------------
Out of the box the app asks you to type the name/price/store. To turn on
automatic reading of those fields FROM the photo, on-device:

  1. Download the official Tesseract.js build files:
       - tesseract.min.js   (replace the small shim of the same name)
       - the worker + core (worker.min.js, tesseract-core.wasm.js)
       - the English data file (eng.traineddata.gz)
     Get them from the tesseract.js project release/CDN on a computer.
  2. Create a folder named  tess  inside the buylater folder and place the
     worker, core, and eng.traineddata.gz files there.
  3. Replace the placeholder tesseract.min.js with the real library file.
  4. Re-host / re-open. Auto-scan now works — and because the app points
     Tesseract at the local  tess/  folder, the model loads from your phone
     only. Your photo is never uploaded.

If you skip this, everything else works exactly the same; you just fill the
fields in yourself.

HOW TO USE
------------------------------------------------------------
ADD        Tap the + button. Choose Camera, Upload, or paste a product URL.
           Confirm/edit the name, price, store and date, add an optional
           note on WHY you want it, then "Save to Buy Later".

WAIT       The item sits in your List with a countdown. Cheaper items get a
           shorter wait, pricier ones longer (you set the rule in Settings).

REVIEW     When the wait ends, the item appears in the Inbox tab (with a
           badge). Open it and choose:
             • Buy it   — you still want it; marks as bought.
             • Defer    — unsure; resets the cooling-off and asks you again
                          next cycle.
             • Drop it  — the urge passed; marks as dropped (money saved).

TRENDS     The Trends tab charts your buy / drop / defer pattern over the
           last 6 months and shows how much impulse spending you've avoided.

REMINDERS  Turn review reminders on/off in Settings. When on (and installed),
           the app notifies you when items are due. Toggle off anytime.

THEME      Light or dark, switchable in Settings.

BACKUP     Settings → Export backup saves a JSON file. "Erase everything"
           wipes all data from the device.

------------------------------------------------------------
NOTE ON "APK"
------------------------------------------------------------
This is delivered as an installable PWA rather than a raw .apk because a PWA
is auditable plain-text (you can read every line and confirm there's no
backdoor or hidden network call) and installs without needing to enable
"unknown sources". If you specifically need a signed .apk, the same folder
can be wrapped with a tool like Bubblewrap or PWABuilder — but that adds a
build toolchain and a signing step, and the privacy behaviour is identical.
